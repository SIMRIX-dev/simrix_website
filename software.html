Software Portfolio

A collection of focused engineering and simulation tools developed for analysis, research, and real-world engineering workflows.

SIMRIX – Suspension Analyzer

A kinematic and quasi-static suspension analysis tool intended for early-stage design evaluation, geometry comparison, and sensitivity studies.

The software focuses on transparent modelling, explicit assumptions, and engineer-controlled inputs, rather than black-box optimisation.

  • Primary use: Suspension geometry analysis
  • Platform: Windows
  • Latest version: v1.0

SIMRIX – Lap Time Simulator

A vehicle performance simulation tool designed to estimate lap time sensitivity to vehicle parameters, setup changes, and operating conditions.

The tool is intended for comparative studies and trend analysis, rather than absolute lap time prediction, with an emphasis on understanding cause and effect.

  • Primary use: Vehicle performance analysis
  • Platform: Windows / Linux
  • Latest version: v0.9
