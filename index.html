What you can expect from SIMRIX software

A consistent set of principles applied across all tools in the SIMRIX ecosystem.

  • Transparent Models

    Underlying equations, assumptions, and modelling approaches are documented and accessible, avoiding black-box behaviour.

  • Versioned Releases

    Every software release is versioned, archived, and remains available to support reproducibility and controlled updates.

  • Technical Documentation

    Each tool includes user guides, theory notes, and reference documentation appropriate to its scope.

  • Engineering-Focused UX

    Interfaces are designed to expose meaningful parameters and outputs, rather than hiding critical behaviour.

  • Independent Tools

    Each SIMRIX application is maintained independently, allowing focused development, clearer support, and cleaner version control.

  • Long-Term Availability

    Released software and documentation remain accessible over time, supporting legacy projects and long-duration development efforts.
