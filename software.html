Software Portfolio

A collection of focused engineering and simulation tools developed for analysis, research, and real-world engineering workflows.

SIMRIX – Suspension Force Calculator

A kinematic-based tool that uses static force equilibrium based solvers for force and moment analysis of a double wishbone suspension.

The software focuses on transparent modelling, explicit assumptions, and engineer-controlled inputs, rather than black-box optimisation.

SIMRIX – Suspension Designer

Another kinematic-based tool that allows for complete control and the design of double wishbone suspension setups.

The software deals primarily with the design and analysis of double wishbone suspension setups, focusing on relations between inputs and outputs.

SIMRIX – Suspension Trackview

The third kinematic-based solver tool that is primarily used for sensitivity studies and variation analysis.

The software mainly models a given suspension through a set of defined motions, allowing you to view the motion through an entire track. It also allows for generating graphs for sensitivity studies.
